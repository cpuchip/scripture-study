# Study — Nate B Jones on Delegation

## Binding Problem

We've built a delegation pattern (steward + commission + covenant + watching until they obey) over the last several studies. Nate B Jones has a video framework on delegating to AI. We don't yet know if his framework exposes gaps in ours, or if ours is the more developed pattern.

## Source

- Video: https://youtu.be/fyHnGHxGuhI
- Compare against: study/art-of-delegation.md, study/art-of-presidency.md, .spec/covenant.yaml, agents using the steward pattern.

## Success Criteria

A study document in study/ai/ (or study/) that:
1. Summarizes Nate's framework faithfully (not strawmanned).
2. Maps it onto our existing pattern (steward, commission, watching, covenant).
3. Names what we'd adopt and what we'd reject (and why).
4. Outputs at least one concrete improvement to our agent instructions.

## Phase 1

Run yt-mcp on the video. Skim transcript. Decide if it warrants a full study or just a paragraph in art-of-delegation.md.
