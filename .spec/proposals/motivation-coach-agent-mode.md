# Motivation / Procrastination Coach Agent Mode

## Binding Problem

Source video: https://youtu.be/tnVI3AFWPJw

There are stuck moments — low motivation, procrastination, the "dream big but do nothing" pattern Michael called out elsewhere. An AI coach with context on what's actually in flight (via brain) could ask the right questions instead of generic ones, and help break the stuck loop into the smallest next step.

## Success Criteria

- yt-mcp transcript analysis of the source video.
- A new agent mode (or sub-skill) that:
  1. Pulls current `.mind/active.md` to know what's in flight.
  2. Asks coaching questions calibrated to where Michael is stuck.
  3. Does NOT default to motivational fluff.
  4. Outputs one concrete, sized-down next step.
- Test: invoke during a real stuck moment, see if it helps.
