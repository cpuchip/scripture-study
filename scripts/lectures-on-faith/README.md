# Lectures on Faith — Scraper

Scrapes all 7 Lectures on Faith (plus introduction and preface) from lecturesonfaith.com and converts them to markdown.

Output: `books/lecture-on-faith/` with files `00_introduction.md` through `07_lecture-7.md`.

## Usage

```bash
go run main.go
```

One-time utility. Results are already committed to the repo.
