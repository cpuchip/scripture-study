# Pearl of Great Price Walk

A start-to-finish walk through the Pearl of Great Price — 16 chapters across 5 books (Moses, Abraham, Joseph Smith–Matthew, Joseph Smith–History, the Articles of Faith) — recording Opus's own thoughts and connections per chapter, using the workspace tools to surface links, building a knowledge graph we can pull from.

Walk #2 of the canon-walk series (the [Book of Mormon walk](../bom-walk/) was #1). Same scaffold, same per-chapter loop, same honesty/cost/resilience frames — see [`_workflow.md`](_workflow.md). Authorized by Michael 2026-06-14 ("Its time for the Pearl of Great Price scripture walk, like we did for the bom walk… commit at every chapter, but don't push").

## What's here

| File | Role |
|------|------|
| `_workflow.md` | the per-chapter rigor + PoGP-specific notes — reloaded at each book boundary and after compaction |
| `_progress.md` | chapter tracker + NEXT pointer — the resume anchor |
| `_journal.md` | Opus's running standout reflections — the narrative thread |
| `_graph.md` | the knowledge-graph index — nodes + edges |
| `{book}/{book}-{NN}.md` | per-chapter digestion notes (structured; also serve as scratch provenance) |
| `studies/` | spin-off full studies (subagent) when a chapter earns more than a note |

## How to read it

The per-chapter notes are the digestible artifact: *binding idea · read · thoughts · connections · entities · edges · notable*. The journal is the personal thread. The graph is for pulling connections — across the PoGP, back into the BoM walk, and into our existing studies.

## The honesty frame

These are **exploratory digestion notes — clearly Opus's own, never doctrine.** Speculation is labeled. Every quote is verified verbatim against the source chapter. This is bin-1/2 gathering under `stuffy-in-the-loop`, not published studies — nothing here is finished doctrinal work without Michael's ratification.
