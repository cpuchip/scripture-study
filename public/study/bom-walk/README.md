# Book of Mormon Walk

A start-to-finish walk through the Book of Mormon — 239 chapters, 15 books — recording Opus's own thoughts and connections per chapter, using the workspace tools to surface links, building a knowledge graph we can pull from.

Authorized by Michael 2026-06-01 as a "100% you" standing effort. Full plan: [`../../.spec/proposals/bom-walkthrough.md`](../../.spec/proposals/bom-walkthrough.md).

## What's here

| File | Role |
|------|------|
| `_workflow.md` | the per-chapter rigor — reloaded at each book boundary and after compaction |
| `_progress.md` | chapter tracker + NEXT pointer — the resume anchor |
| `_journal.md` | Opus's running standout reflections — the narrative thread |
| `_graph.md` | the knowledge-graph index — nodes + edges |
| `{book}/{book}-{NN}.md` | per-chapter digestion notes (structured; also serve as scratch provenance) |

## How to read it

The per-chapter notes are the digestible artifact: *binding idea · read · thoughts · connections · entities · edges · notable*. The journal is the personal thread. The graph is for pulling connections — across the BoM, and into our existing studies.

## The honesty frame

These are **exploratory digestion notes — clearly Opus's own, never doctrine.** Speculation is labeled. Every quote is verified verbatim against the source chapter. This is bin-1/2 gathering under `stuffy-in-the-loop`, not published studies — nothing here is finished doctrinal work without Michael's ratification.
