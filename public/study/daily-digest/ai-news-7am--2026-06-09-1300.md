**Apple unveils a next-generation Siri with cross-app personal context.** Announced at WWDC26 (June 9, 2026), Apple introduced Siri AI integrated across iPhone, iPad, Mac, Apple Watch, and Vision Pro. It searches across messages, emails, and photos using personal context, performs systemwide app actions, and adds on-screen awareness plus web lookup. Developer betas for iOS 27, iPadOS 27, macOS 27, and visionOS 27 are available immediately; public beta next month, general release this fall. [Apple unveils next generation of Apple Intelligence, Siri AI, and more](https://www.apple.com/newsroom/2026/06/apple-unveils-next-generation-of-apple-intelligence-siri-ai-and-more/)

**Google upgrades NotebookLM with agentic chat and a secure cloud computer.** Announced June 9, 2026. Running on Gemini 3.5, the new release adds an agentic chat interface, a secure cloud computer that writes and runs code across more than 100 curated software skills, and multi-format export including charts, PDF, Excel, and PPT. A new mode starts from a loose idea and auto-discovers web sources. Google cites internal side-by-side win rates of 65% average and 78.2% on web research against its own prior baseline. Rollout is limited to Google AI Ultra and select Workspace business tiers for now. [Do better research with NotebookLM](https://blog.google/innovation-and-ai/products/notebooklm/better-research-notebooklm/)

**MetaMask opens early access to a self-custodial Agent Wallet for AI agents.** Announced June 9, 2026. The CLI wallet lets users set operating policies before an agent transacts across EVM chains and Hyperliquid. It includes mandatory 2FA on flagged transactions, Guard and Beast operating modes, and up to $10,000/month in Transaction Protection. Supported frameworks include Codex, Claude Code, Cursor, and Copilot. [Introducing MetaMask Agent Wallet](https://metamask.io/news/introducing-metamask-agent-wallet)
