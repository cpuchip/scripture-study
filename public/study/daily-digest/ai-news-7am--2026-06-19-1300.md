# AI Daily Digest — June 19, 2026

## Headlines

**Mistral 3 family ships under Apache 2.0.** Mistral released the Mistral 3 family, headlined by Mistral Large 3 — a 675B-parameter sparse MoE with only 41B active parameters — plus three Ministral 3 edge models (3B, 8B, 14B) [Mistral 3 Release](https://tpsreport.news/news/mistral-3-release-large-675b-ministral-models). Apache 2.0 licensing on a model this large is notable: it's the biggest permissively-licensed open-weight model to date. Pricing is $0.50/$1.50 per 1M input/output tokens for Large 3. Mistral claims it ranks #2 among open-source non-reasoning models on LMArena. A reasoning variant is planned but not yet released.

**DeepSeek V4-Pro: 1.6T parameters, million-token context.** DeepSeek released V4-Pro (1.6T total, 49B activated) and V4-Flash (284B total, 13B activated), both supporting 1M-token contexts [DeepSeek-V4-Pro](https://huggingface.co/deepseek-ai/DeepSeek-V4-Pro). The architecture uses a hybrid attention mechanism (CSA + HCA) that cuts single-token inference FLOPs to 27% and KV cache to 10% versus V3.2 at 1M context. Pre-trained on 32T tokens. The Max reasoning mode claims to be the best open-source model available today, though independent benchmarking is still pending.
