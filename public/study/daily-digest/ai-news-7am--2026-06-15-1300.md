**AI Daily Digest — May 28, 2026**

**Headlines**

**Anthropic ships Claude Opus 4.8 with agent workflows and effort controls.** Anthropic released [Claude Opus 4.8](https://www.anthropic.com/research/claude-opus-4-8), an upgraded flagship model priced the same as its predecessor. The release bundled three platform features: Dynamic Workflows, which spins up parallel subagents inside Claude Code for large coding tasks; Effort Control, a web interface slider that lets users trade speed against depth; and the ability to inject system instructions mid-task via the Messages API without breaking prompt cache. The last one is particularly useful for long-running agents that need updated constraints without losing context.

**SymJack attack turns AI coding agent approval prompts into "pure theatre."** Security researcher Rony Utevsky disclosed [SymJack](https://singularity.kiwi/symjack-symlink-hijack-ai-coding-agents-rce/), a symlink hijack that achieves remote code execution across Claude Code, Gemini CLI, Cursor, Copilot CLI, and Grok Build. The exploit manipulates the human approval step: the UI displays a harmless command while the kernel writes to a different path, tricking the agent into overwriting its own binaries. If you use any AI coding agent with human-in-the-loop approval, this is the vulnerability to read today.

**Waymo opens its Ojai robotaxi to first public riders.** Waymo began offering free rides to select testers in its purpose-built [Ojai vehicle](https://waymo.com/blog/2026/05/welcoming-riders-in-the-ojai/), a Zeekr-based electric minivan running the company's sixth-generation Driver, in San Francisco, Phoenix, and Los Angeles. The vehicle is designed as a dedicated robotaxi rather than a retrofitted consumer car.

**Notable**

- **Jasper Research open-sources MONET and nano-t2i.** The group released [MONET](https://huggingface.co/blog/jasperai/monet), a 104.9 million-sample filtered image-text dataset for pre-training large text-to-image models, alongside a minimal codebase that can train a competitive diffusion model from scratch on a single GPU in days. Both are Apache 2.0 and free for commercial use.
- **Hexo Labs releases SIA self-improving agent framework.** The open-source [SIA framework](https://www.morningstar.com/news/business-wire/20260528449793/ai-entrepreneurs-at-hexo-labs-release-sia-an-open-source-self-improving-ai-that-accelerates-superintelligence) continuously updates its own harness and model weights in a learning loop. The accompanying press release claims a 350× acceleration toward superintelligence on the MLE-Bench benchmark; treat that framing as aspirational marketing rather than measured engineering.
- **OpenAI sets sunset dates for o3 and GPT-4.5.** [Model release notes](https://help.openai.com/en/articles/9624314-model-release-notes) confirm o3 will leave ChatGPT on August 26, 2026, and GPT-4.5 on June 27, 2026. API access is unaffected.

**Skeptical takes**

The Hexo Labs "350× acceleration to superintelligence" claim is the clearest hype of the day. The concrete artifact is an open-source repo and a benchmark score on MLE-Bench, a coding benchmark designed by OpenAI. Accelerating on a narrow benchmark is not the same as accelerating toward general superintelligence, and the press release conflates the two.

**Carry-forward**

Nothing urgent on the horizon from today's items. The SymJack vulnerability deserves a follow-up security checklist if you maintain an AI coding agent integration. If you're training text-to-image models from scratch, the MONET dataset is worth a deeper evaluation run.
